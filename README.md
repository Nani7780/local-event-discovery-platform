# 📍 Local Event Discovery Platform 🚀

A high-density, lightning-fast web application built to eliminate community event fragmentation by connecting residents with local workshops, professional seminars, cultural gatherings, and tech career job fairs.

---

## 💡 The Problem & Our Solution
* **The Problem:** Valuable local workshops, career milestones, and community seminars are frequently missed due to fragmented, outdated, or scattered messaging channels.
* **Our Solution:** A single, consolidated event discovery hub featuring automated layout grid tracking feeds, integrated localized calendar synchronization parameters, and a real-time responsive QR code ticket verification ecosystem.

---

## 🛠️ Tech Stack & Systems Architecture
* **Frontend UI Layout:** React (Vite environment), utilizing responsive structural CSS layouts.
* **Backend Core Engine:** Node.js + Express web framework runtime.
* **Security Layer:** Secure password hashing schemas via BcryptJS + JSON Web Tokens (JWT).
* **Data Layer:** High-speed In-Memory Array Data Store (Engineered to bypass localized cloud firewall network port restrictions for absolute runtime uptime stability).

---

## 📡 API Endpoints Cheat Sheet

Collaborators can target frontend components directly to the active server port listener:

### 🔐 Identity & Authentication
* **POST** `/api/auth/register` — Provisions new user data profiles and assigns structural organizer flags.
* **POST** `/api/auth/login` — Validates user credential signatures and issues secure JWT tokens.

### 📅 Event Management Tracking
* **POST** `/api/events` — Allows verified event organizers to electronically publish upcoming local gatherings.
* **GET** `/api/events` — Dynamically pulls the full active list array of community workshops.

### 📷 Smart Entry Check-In
* **POST** `/api/registrations/check-in` — Matches digital QR payloads against active data structures to verify attendees instantly.

## 🚀 Fast Local Deployment Setup

### 1. Backend Server Initialization
```bash
# Navigate to the backend service core
cd backend

# Assemble your tracking dependency tree nodes
npm install

# Launch the live API service listener on port 5000
node server.js
```

### 2. Frontend Interface Run
```bash
# Move into the user interface directory 
cd frontend

# Install package modules
npm install

# Fire up the reactive Vite development platform
npm run dev
```
